Opciones

Personaliza las alertas, cambiando el mensaje, tiempo de duración, el color y si el mensaje sera permanente.

Opción Tipo Default Descripción
text Text | HTML Hello World Mensaje
type warning | success | danger | info warning Tipo de mensaje
time Number | Seconds 5 Tiempo que dura la alerta
permanent Boolean false La alerta es permanente
