Que incluye

Una vez descargado el archivo, descomprime el .zip para ver la estructura de Smoke. Verás algo como esto:

smoke/
├── css/
│   ├── smoke.css
│   └── smoke.min.css
└── js/
   ├── smoke.js
   └── smoke.min.js

Requerimientos

El plugin Smoke requiere que se incluyan estas dos librerías para su funcionamiento.

jQuery requerido
Ten en cuenta que el plugin Smoke requiere que se incluya la librería jQuery.
Bootstrap 3 requerido
PTen en cuenta que el plugin Smoke requiere que se incluya el framework Bootstrap 3.

Plantilla básica

Comienza con esta plantilla de HTML o modifícala a tus necesidades.

Copia el código HTML de abajo para empezar a trabajar con los requerimientos mínimos de Smoke.

<!DOCTYPE html>
<html lang="es" >
  <head>
    <meta charset="utf-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1">
    <title>Plantilla básica</title>

    <!-- Bootstrap (necesario para el CSS del plugin Smoke) -->
    <link href="css/bootstrap.min.css" rel="stylesheet">
    <!-- Smoke -->
    <link href="css/smoke.min.css" rel="stylesheet">

    <!-- HTML5 shim and Respond.js for IE8 support of HTML5 elements and media queries -->
    <!-- WARNING: Respond.js doesn't work if you view the page via file:// -->
    <!--[if lt IE 9]>
      <script src="https://oss.maxcdn.com/html5shiv/3.7.2/html5shiv.min.js"></script>
      <script src="https://oss.maxcdn.com/respond/1.4.2/respond.min.js"></script>
    <![endif]-->
  </head>
  <body>
    <h1>Hola Mundo!</h1>

    <!-- jQuery (necesario para el Javascript del plugin Smoke) -->
    <script src="https://code.jquery.com/jquery.js"></script>
    <!-- Bootstrap (necesario para el Javascript del plugin Smoke) -->
    <script src="js/bootstrap.min.js"></script>
    <!-- Smoke -->
    <script src="js/smoke.min.js"></script>
  </body>
</html>
